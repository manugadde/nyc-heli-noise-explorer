# NYC Helicopter Noise Explorer

Showcase application for the [ArcGIS Maps SDK for JavaScript](https://js.arcgis.com/) that explores helicopter activity, noise complaints, and census context across New York City.

## Prerequisites

- Node.js 19+ (for npx)
- A modern desktop browser

## Development

1. Start a local development server.
   ```bash
   npx serve . -l 8080
   ```

   Alternative:
   ```bash
   npx http-server -p 8080
   ```

2. Open the application in your browser.
   ```text
   http://127.0.0.1:8080/
   ```

3. Update the source files as needed.
   ```text
   index.html
   src/style.css
   src/js/
   ```

## Deployment

GitHub Pages deployment is configured with a workflow in `.github/workflows/pages-deploy.yml`.
